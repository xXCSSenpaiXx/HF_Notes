## Basic
- **clear**
    - clear screen and variables / values

    - only clear screen
- **result lenght**
    - format long
    - format short

## Files
- **save**
    - save .mat file
    - save filename

- **load**
    - load .mat file
    - load filename
    - load filename variable - only load defined variable
