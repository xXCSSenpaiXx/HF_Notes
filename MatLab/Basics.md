# Commands

## Basic
- **clear**
    - clear screen and variables / values
- **clc**
    - only clear screen
- **result lenght**
    - format long
    - format short

## Files
- **save**
    - save .mat file
    - save filename
    - save filename variable - only save defined variable
- **load**
    - load .mat file
    - load filename
    - load filename variable - only load defined variable

# Data Types
## Scalar
Code:
```
    x = 2
```
Example:
| 2        |
|--------------|

## Row Vector
Code:
```
    x = [ 2 -9 5 ] 
```
Example:
| 2        | -9         | 5                    |
|--------------|---------------|--------------------------|

## Column Vector
Code:
```
    x = [ 2; 4; 3; ] 
```
Example:
| 2        |
|--------------|
| 4         | 
| 3         | 
Code:
```
    x = [ 2 -9 5; 4 23 11; 3 5 -6; ] 
```
